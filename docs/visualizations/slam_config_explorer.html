inflation_radius Nav2 costmap
(inflation layer)		 SOFT cost
(decaying gradient outside robot_radius)		 "How much should I prefer to stay away?" Outside the lethal zone, costs decay smoothly via exp(-K·(d − robot_radius)). Watch as you slide: the yellow dashed ring (range_min) discards returns near the rover before they enter the map; the cyan solid ring (robot_radius) is a hard wall the path can't cross; the red gradient (inflation) bends the path toward open space without forbidding closer cells.
🔗 They stack, they don't replace each other: a sensor return survives range_min → it becomes an obstacle cell in the costmap → that cell gets a lethal ring of width robot_radius → which then gets an inflation gradient of width inflation_radius on top. So the total "no-go" distance from a true obstacle is robot_radius + inflation_radius (with the inner part hard-forbidden and the outer part soft-penalized).

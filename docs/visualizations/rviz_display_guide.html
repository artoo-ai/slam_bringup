RViz / Foxglove Display Guide

Every checkbox in the RViz Displays panel (or panel type in Foxglove) renders one kind of data from your robot's ROS topic tree. This page maps all 10 display types used across the five .rviz configs in this project, explains what each one shows and why, and arranges them in a vertical flow from raw sensor data at the top down through derived perception and planning/navigation at the bottom.

Click any node in the diagram for full details — topics, QoS, when to enable it, and the Foxglove equivalent.

Which Displays Are On In Each Config?

ON = enabled by default when you open this .rviz file. off = present in the file but toggled off — you can enable it in the Displays panel. — = not in that config at all.
